## 🔧 技術實現

### 1. **數據結構**
```typescript
interface Work {
  id: string;
  title: string;
  type: string;
  status: "進行中" | "已完結" | "暫停" | "放棄";
  year?: number;
  rating?: number;
  review?: string;
  source?: string;
  episodes: Episode[];
  tags: Tag[];
}
```

### 2. **初始化流程**
1. 檢查現有數據
2. 如果沒有數據，創建 demo 作品
3. 創建 demo 標籤
4. 更新統計信息

### 3. **數據管理**
- **IndexedDB**: 使用 IndexedDB 存儲數據
- **統計計算**: 實時計算統計信息
- **數據同步**: 支援數據匯入/匯出
